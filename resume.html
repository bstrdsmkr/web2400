Projects

2014 - Present

Linux Loader

Oak Ridge National Lab

The Linux Loader is a project to convert Linux desktop installations from an IT cost center to a self-service affair, saving an estimated 300 man hours per year. This is an automated deployment system which allows end users to pxe boot to a Cobbler server which supplies a kickstart profile based on the user's selection in the pxe menu. The supplied kickstart profile partitions attached disks, provisions user accounts, and sets up compliance and auditing systems without user intervention.
